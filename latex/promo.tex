\heading{1.\quad Built for real formulas}

The same source stays readable while the font handles spacing and stretch:
\[
  \left( \frac{1}{2}+\frac{x}{y} \right),\qquad
  \binom{n}{k}=\frac{n!}{k!(n-k)!},
\]
\[
  f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h},
\]
\[
  i h\frac{\partial}{\partial t}\Psi(x,t)=
  \left[-\frac{h^2}{2m}\frac{\partial^2}{\partial x^2}+V(x,t)\right]\Psi(x,t).
\]

\heading{2.\quad Stretchy pieces stay pixel-sharp}

Roots, delimiters, and large operators are driven by the OpenType MATH table:
\[
  \sqrt{2},\quad
  \sqrt{x^2+y^2},\quad
  \sqrt{1+\sqrt{1+\sqrt{1+x}}},\quad
  \left|\sum_{k=0}^{\infty}\frac{1}{2^k}\right|=2.
\]

\heading{3.\quad Good for PDFs and demos}

Use the same visual language in printable notes, posters, README previews, and
browser demos with Temml-generated MathML:
\[
  \int_{-\infty}^{\infty}\frac{1}{\sqrt{2\pi}}e^{-x^2/2}\,dx=1,
  \qquad
  \prod_{i=1}^{n}i=n!.
\]

\heading{4.\quad Matrices, systems, and relations}

\[
  A=\begin{pmatrix}a&b\\c&d\end{pmatrix},\qquad
  A^{-1}=\frac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix},\qquad
  |x|=\begin{cases}\phantom{-}x,&x\geq0,\\-x,&x<0.\end{cases}
\]
\[
  \alpha+\beta=\gamma,
  \qquad \theta\in[0,2\pi),
  \qquad \nabla\cdot F=\rho,
  \qquad a\leq b\approx c\neq d.
\]

Dense formulas still come from ordinary LaTeX source:
\[
  p(x)=\frac{1}{\sqrt{(2\pi)^n|\Sigma|}}
  \exp\!\left(-\frac{1}{2}(x-\mu)^T\Sigma^{-1}(x-\mu)\right).
\]
\[
  \left\{
  \begin{aligned}
    \frac{\partial u}{\partial t} &= \nabla\cdot(D\nabla u)+f(u),\\
    u(x,0) &= u_0(x),\qquad u|_{\partial\Omega}=0.
  \end{aligned}
  \right.
\]

\heading{5.\quad What the project includes}

\begin{itemize}
  \item A pure Python font build pipeline using fontTools, no FontForge needed.
  \item A LaTeX package for LuaLaTeX and XeLaTeX through unicode-math.
  \item A small web wrapper: LaTeX source to Temml MathML to native browser math.
  \item MIT tooling; OFL font derived from Monocraft.
\end{itemize}
